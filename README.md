Raspberry Pi Pinout
-------------------

| Pin | Input/Output | Function     |
| --- | ------------ | ------------ |

| 22  | Output       | LED2 - Green |
| 23  | Output       | LED1 - Red   |
| 24  | Output       | Switch Left  |
| 25  | Input        | Go Button    |
